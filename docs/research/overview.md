# 校内研究の概要

## 研究主題

**学び方を選択し自己調整する児童の育成**
～算数科におけるマイプラン型学習の工夫を通して～

## 研究仮説

単元内にチェックポイント（CP：到達確認）を位置付け、結果に基づいて学び方を選び直すことを保障し、ふり返りで学び方と成果を言語化させるとともに、教師がミニレッスン等の支援を行えば、児童の自己調整（計画・実行・調整・意欲）は高まり、自ら進んで学習する児童が育つであろう。

## めざす児童像

「今日のめあては何か」「どんなやり方が自分に合うか」「分からなくなったらどうするか」を自分で考え、必要に応じて学び方を選び直したり、友達や教師の力を借りたりしながら自己調整力を発揮して主体的に学習を進める児童。

## 研究の3つの視点

| 視点 | 内容 |
|------|------|
| **見通し** | 単元のゴールと学習の道筋が分かる資料 |
| **選択肢** | 学び方の選択（例題・ヒントカード・ICT・相談先・ミニ指導） |
| **確認** | チェックポイント（到達確認）と振り返り（次の作戦づくり） |

## マイプラン型学習とは

単元のゴールと学習の道筋を示した上で、児童が「今日は何をどこまで」「どの方法で学ぶか」を自分で決めて進める学び方。途中でつまずいたら、方法を選び直し、チェックポイントで確かめる。

## 年間スケジュール

| 月 | 内容 |
|----|------|
| 5月 | 年間の大まかな見通し（学年研） |
| 6月 | 理論研（講師招聘検討）、教材研究 |
| 7月 | 教材研究 |
| 9月 | 研究授業（中学年） |
| 11月 | 研究授業（**高学年** ← 5・6年関連）、教材研究 |
| 12月 | 研究授業（低学年）、実践報告準備 |
| 2月 | **実践報告会** |
