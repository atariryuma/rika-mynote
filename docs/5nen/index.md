## このページの使い方

!!! tips "効果的な活用"

    1. **授業準備**: 該当単元の「Plan」をクリックして教材研究を確認
    2. **授業実施**: 「Do&Check」で授業記録を参照
    3. **授業改善**: 「Action」で前回の改善点を確認
    4. **学習評価**: 各単元の到達度チェック項目を確認
    5. **次年度準備**: 「Action」で継続的改善を反映
