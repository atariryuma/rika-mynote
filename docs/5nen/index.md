# 5年生の理科学習

10単元、計105時間で学習します。各単元では問題解決の過程で「見通し→実行→振り返り」を繰り返し、科学的思考力の段階的な育成を目指します。

## 学習進度状況

!!! info "2026年4月 現在地"

    **理科開き（rika-biraki）** が進行中です。
    
    この1時間で、5年生からの理科学習全体の見通しを持ち、問題解決の基本的な流れを体験します。次は「天気の変化」へ進みます。

---

## 学習の進め方

各単元は以下の3つのステップで構成されています：

!!! note "3つのステップ"

    1. **Plan（計画）**  
       単元の全体像と学習計画を立てる
    
    2. **Do & Check（実行と確認）**  
       観察・実験を行い、記録と検証を進める
    
    3. **Action（振り返り）**  
       学習を振り返り、次の課題へつなぐ
