## 6年生の特徴

!!! info "5年生からの発展"

    6年生では以下の点が重視されます：
    
    - **多面的思考**: 1つの現象から複数の視点で考察
    - **複合的問題解決**: 複数の単元の知識を統合
    - **自立的学習**: 児童が計画を立案し実行する機会の拡大
    - **環境への視点**: 持続可能な社会への問題意識

## このページの使い方

!!! tips "効果的な活用"

    1. **授業準備**: 該当単元の「Plan」をクリックして教材研究を確認
    2. **授業実施**: 「Do&Check」で授業記録を参照
    3. **授業改善**: 「Action」で前回の改善点を確認
    4. **学習評価**: 各単元の到達度チェック項目を確認
    5. **次年度準備**: 「Action」で継続的改善を反映
    6. **指導要領確認**: リソース集の「指導要領対応表」で学習内容の対応を確認
