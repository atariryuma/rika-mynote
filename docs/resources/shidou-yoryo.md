# 学習指導要領対照表

**小学校理科 5年・6年 学習指導要領との対応**

---

## 学習指導要領の概要

!!! info "文部科学省『小学校学習指導要領』（平成29年告示）"
    小学校理科は、4つの領域で構成されます：
    
    - **A物質・エネルギー**: 物質の性質・変化、エネルギー現象
    - **B生命・地球**: 生命現象、地球現象
    - **C様々な現象の観察・実験**: 実践的スキル

本校では、A領域とB領域を中心に指導計画を構成しています。

## 5年・6年を通じた学習の階層性

### 学習内容のつながり

```
5年: 基礎的な現象の理解 → 6年: 複雑な現象・統合的理解
```

| 領域 | 5年生での学習 | 6年生への発展 |
|------|--------------|-------------|
| **物質** | 溶解（1物質が水に） | 複数物質の性質比較（酸性・アルカリ性） |
| **生命** | 個別の成長過程（植物・動物） | 複数生物の相互作用（食物連鎖） |
| **地球** | 気象変化・流水作用 | 地層形成・地殻変動 |
| **エネルギー** | 単一の力（磁力） | 複合的利用（電気） |

### 問題解決スキルの発展

| 段階 | 5年生 | 6年生 |
|------|------|------|
| **予想** | 単一の予想 | 複数の仮説を立案 |
| **実験計画** | 示された条件から選択 | 独立して計画を立案 |
| **結果考察** | 予想と結果を比較 | 多角的視点から検証 |
| **応用** | 学んだことを記述 | 新しい問題解決に応用 |

## 研究主題との関連

!!! success "自己調整学習への対応"

    ### 見通し（前主体的に学ぶために）
    - 各単元の学習指導要領上の位置づけを明確化
    - 5年→6年への学習の連続性を提示
    - 「このことを学ぶ意味」を児童に理解させる
    
    ### 選択肢（複数の方法を提供）
    - 学習指導要領の基本内容は共通
    - 学習方法は複数の選択肢から児童が選択可能
    - 例）「水溶液の性質を調べる」→ BTB液・リトマス試験紙など複数検査法
    
    ### 振り返り（到達度の確認）
    - 学習指導要領に示された内容の定着を確認
    - 定着が不十分な場合は補充学習
    - 深化させたい児童には発展学習を提供

## 指導計画の活用方法

!!! tips "この対照表を活用する際"

    1. **単元設計時**: 学習指導要領との対応を確認
    2. **授業準備時**: 単元の学習内容の網羅性をチェック
    3. **評価時**: 指導要領に示された内容が定着しているか確認
    4. **カリキュラム改善時**: 学年間・教科間の連携を検討
    5. **保護者説明時**: 「何をどこまで学ぶか」を明確に説明
