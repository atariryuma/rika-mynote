## よくある質問（FAQ）

!!! faq "Q: 指導計画に時数が異なる場合があるのはなぜですか？"
    
    A: 括弧内の時数は実施状況に応じた調整時数です。学級の進度や季節的要因により、柔軟に時数を配分しています。

!!! faq "Q: 「研究の重点」の視点で授業を設計するには？"
    
    A: 各単元の「研究の重点」を確認し、該当する視点（見通し・選択肢・振り返り）の手立てを授業に組み込んでください。詳細は「校内研究の概要」をご覧ください。

!!! faq "Q: 各単元の詳しい教材研究資料はどこにありますか？"
    
    A: 5年・6年のリソース欄から各単元を選択し、「Plan」のページで確認できます。

!!! faq "Q: 学習指導要領との対応を確認したいのですが？"
    
    A: 「学習指導要領対照表」で A領域・B領域別に対応を示しています。また、各単元の Plan でも指導要領との関連を記載しています。

## サイトの活用方法

### 👨‍🏫 教師向け

```
1. 年度初めに「年間指導計画」で全体を把握
2. 各単元ごとに「学習指導要領対照表」で位置づけを確認
3. 教材研究時に「校内研究の概要」で視点を確認
4. 授業準備時に該当単元の「Plan」で詳細を確認
5. 授業後に「Do & Check」で記録
6. 単元終了後に「Action」で改善を検討
```

### 📊 カリキュラム全体の把握

```
年間指導計画（全体像）
        ↓
各単元の指導要領対応（細目確認）
        ↓
校内研究の視点（教育的価値）
        ↓
各単元の実践（実装）
```
