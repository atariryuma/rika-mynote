# 6年 理科学習指導計画

## 全体概要

6年間の理科学習の集大成として、物質・エネルギー領域と生命・地球領域にわたる13単元で構成されます。

- **総時間数**: 105時間
- **対象クラス数**: 4クラス
- **単元数**: 13単元（導入・まとめ含む）
- **学習期間**: 4月～3月（3学期制）

!!! info "学習目標"
    6年生では、実験計画の選択、調べ方の工夫、多面的な考察、条件制御の自立化など、問題解決能力を段階的に高めていきます。

## 領域別構成

### A領域：物質・エネルギー（計52時間）

物の状態や変化、エネルギーの捉え方を深める。

- **物の燃え方と空気** (8h) - 実験計画の選択
- **水溶液の性質とはたらき** (11h) - 実験方法の自己選択
- **電気と私たちのくらし** (10h) - 総合的問題解決
- **てこのはたらきとしくみ** (10h) - 条件制御の自立
- **動物の体のつくりとはたらき** (8h) - 調べ方の選択
- **植物の体のつくりとはたらき** (7h) - 理解度確認と深化

### B領域：生命・地球（計51時間）

生き物の営みと地球の動きを統合的に理解する。

- **生き物どうしのかかわり** (4h) - 情報収集方法の選択
- **月の形と太陽** (7h) - モデル実験の計画
- **大地のつくり** (8h) - 多面的な考察
- **変わり続ける大地** (4h) - 災害・防災の視点
- **地球と私たちのくらし（導入）** (2h) - 年間の見通し
- **地球に生きる（まとめ）** (2h) - 年間のまとめ
- **理科開き** (1h) - 問題解決の流れ
- **その他** (13h)

## 各単元の学習ページ構成

各単元は3つのページで構成されています：

- **plan.md** - 単元の導入、学習の見通し、実験計画の立案
- **do-check.md** - 実験・観察の実施と結果の記録、検証
- **action.md** - 結果の考察、深める活動、次単元への架橋

!!! tip "学習活動の流れ"
    各単元で「計画 → 実行・記録 → 振り返り・深化」のサイクルを回します。

## 5年からの発展点

6年の学習は5年で培った基礎を土台としています：

- **実験技能の深化** - 5年で習った計測・観察をより精密に
- **見通し能力の発展** - 5年の「単元導入」から「学期・年間見通し」へ拡大
- **調べ方の選択肢化** - 5年までの「決められた方法」から「自分で選ぶ」へ
- **記録・表現の工夫** - データの分類、整理、可視化の高度化
- **多角的思考の確立** - ひとつの現象を複数の視点から捉える力

## このページの使い方

!!! tips "効果的な活用"

    1. **授業準備**: 該当単元の「Plan」をクリックして教材研究を確認
    2. **授業実施**: 「Do&Check」で授業記録を参照
    3. **授業改善**: 「Action」で前回の改善点を確認し、次回の授業設計に反映
    4. **学習評価**: 各単元の到達度チェック項目を確認
    5. **年間視点**: 領域別・学習視点別の分類から、学習の縦横の関連を把握
    6. **指導要領確認**: リソース集の「指導要領対応表」で学習内容の対応を確認
